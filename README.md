## 🚀 Featured Project

### 📚 Rural School Management API

A full REST API to manage rural school operations across three business domains (Institutional, Identity, Academic), with role-based access control and JWT authentication.

**Stack:** Java 21 · Spring Boot 3.5.1 · Spring Security (JWT) · JPA / Hibernate · MapStruct · MySQL · Docker · Gradle · Springdoc OpenAPI

**Highlights:**
- 🔐 JWT-based authentication and authorization with Spring Security
- 🏗️ Domain-driven structure with 13 entities across 4 bounded contexts
- 🐳 Containerized database via Docker Compose
- 📖 Interactive API documentation with Swagger UI
